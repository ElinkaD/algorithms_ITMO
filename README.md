# algorithms_ITMO
Репозиторий содержит решения лабораторных работ по предмету
**«Структуры и алгоритмы в базах данных и распределенных системах» (ИТМО).**

## Первое дз - Hashing Algorithms
### Необходимо реализовать три алгоритма (в скобках операции): 
1) хэш таблицу на файловой системе с бакетами как мы обсуждали на паре (вставка, обновление и удаление) 
2) perfect hash (создание полного индекса по фиксированному набору ключей и поиск) 
3) lsh для поиска дублей в текстах или точек в 3d (создание индекса, добавление новых точек, фулл скан для поиска дублей) 

### По всем алгоритмам должны быть: 
* тесты функциональности на случайных наборах данных (набор данных при тесте генерируется) - очень помогает ловить корнер-кейсы 
* перфтесты на больших объемах данных - считаем пропускную способность и время операции 
* профайлинг памяти и цпу всех реализаций с анализом узких мест и набором гипотез как можно сделать лучше 

## Результат

На выходе должен быть репозиторий, содержащий:

* исходный код реализаций
* тесты
* результаты перфоманс-тестов
* профили CPU и памяти
* отчет с анализом полученных результатов

## Второе дз - Геопоиск по Lat/Lng
### Что должна уметь система:
1) хранить географически расположенные объекты
2) выполнять поиск объектов по координатам
3) находить не только точные совпадения, но и ближайшие объекты в заданной области или с заданной точностью

### В работе также требуется:
* реализовать структуру данных или индекс для геопоиска по координатам `Lat/Lng`
* подготовить функциональные тесты для размещения и поиска объектов
* написать бенчмарки и показать:
  либо сравнение выбранного алгоритма с разумными аналогами,
  либо зависимость времени работы от размера данных, параметров запроса и точности поиска

## Результат по второй лабораторной работе

На выходе по второй лабораторной работе должен быть репозиторий, содержащий:

* исходный код реализации
* тесты
* результаты бенчмарков
* краткий отчет с анализом полученных результатов
